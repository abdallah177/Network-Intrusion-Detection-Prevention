// Overview

What is NetGuard?

NetGuard IDS/IPS is a network security solution that monitors, analyzes, and protects computer networks from unauthorized access in real time. It integrates Intrusion Detection and Intrusion Prevention using signature-based and anomaly-based techniques. It also features an Arduino-based hardware LED indicator — white LED for active monitoring (1s blink via pin 13), red LED for detected attacks (200ms rapid blink via pin 2). Built to simulate enterprise-level cybersecurity while ensuring the CIA triad.

// Arduino LED Integration

💡 LED Blinking Indicator

NetGuard integrates an Arduino Uno as a physical network status display. The Arduino receives serial commands from NetGuard and controls LEDs using digitalWrite() — setting pins HIGH (5V on) and LOW (0V off) with timed delays to create the blink pattern.

⚠️  Important: Connect the Arduino to USB Port 3. Each LED needs a 220Ω resistor in series — connect the longer leg (anode +) through the resistor to the digital pin, and the shorter leg (cathode −) to GND. Skipping the resistor will burn out the LED instantly.
White LED — Monitoring Active
Blinks every 1 second when "Start Monitoring" is pressed. Arduino sets pin HIGH (5V) for 1000ms then LOW — confirming the system is live and scanning all traffic.
Pin 13delay: 1000msHIGH → LOW
Red LED — Attack Detected!
Blinks rapidly every 200ms when a threat is identified — SYN flood, UDP flood, ICMP flood, or port scan. Fast blink signals urgency for immediate action.
