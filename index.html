// Overview

What is NetGuard?

NetGuard IDS/IPS is a network security solution designed to monitor, analyze, and protect computer networks from unauthorized access and malicious activities in real time. The system integrates both Intrusion Detection and Intrusion Prevention capabilities, using signature-based and anomaly-based detection techniques. It also features a physical Arduino-based LED circuit that gives real-world visual feedback — a white LED blinks during monitoring and a red LED blinks on attack detection.

Project Goals

// Hardware Integration

💡 LED Blinking Indicator

NetGuard bridges software and hardware — an Arduino circuit provides a physical, real-world signal of your network's status so you always know what's happening, even without looking at the screen.

⚠️  Important: Connect the Arduino to USB Port 3 on your laptop or PC. NetGuard's serial communication is configured specifically for COM3 (Port 3) to send LED control signals to the Arduino.
White LED — Monitoring Active
Blinks steadily when you press "Start Monitoring". Signals the system is live and actively scanning all network traffic.
Red LED — Attack Detected!
Blinks rapidly the moment a threat is identified — SYN flood, port scan, UDP flood, or any malicious pattern.
